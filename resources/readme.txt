Встановлення git  на комп’ютер. Деталі: https://git-scm.com/book/ru/v2/Git
Перше, що вам слід зробити після установки Git - вказати ваше ім'я та адресу електронної пошти. Це важливо, тому що кожен Коміт в Git містить цю інформацію, і вона включена в комітів, що передаються вами, і не може бути далі змінена:
$ git config --global user.name "_username_"
$ git config --global user.email _email_

Знову ж таки, якщо вказана опція --global, то ці настройки досить зробити тільки один раз! 

1. Створити проект (із правильною структурою)
2. Ініціалізувати Git

   надіслати html версію на ваш віддалений репозиторій github.com.

5. Надіслати мені в особисті повідомлення адреси своїх репозиторіїв.
